## 🚀 現在のプロジェクト

### [TopFlow](https://github.com/csupenn/topflow)（開発中）
安全なAIアプリケーションのためのオープンソースビジュアルワークフロービルダー。

**主な機能：**

- AI オーケストレーション向けの 12 種類のノード
- セキュリティ重視のテンプレート（GDPR、SOC 2、HIPAA）
- SSRF 対策、レート制限、循環検出
- プライバシーファーストなアーキテクチャ（localStorage のみ使用）
- TypeScript コードへのエクスポート
